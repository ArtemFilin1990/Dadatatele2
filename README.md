# Dadatatele2

`Dadatatele2` — репозиторий с Telegram-ботом для проверки ИНН через DaData API.

## Что находится в репозитории
- `inn_checker_bot/` — основной Python-проект бота.
- `LICENSE` — лицензия MIT.

## Назначение бота
Бот принимает ИНН от пользователя, валидирует его (формат и контрольные суммы),
после чего запрашивает данные организации/ИП в DaData и возвращает результат в чат.

Поддерживаются 2 режима поиска:
1. **Прямой DaData API** — реальный HTTP-запрос к DaData.
2. **OpenAI + MCP** — в текущей реализации это безопасный fallback на прямой DaData-клиент
   (интерфейс под MCP сохранён, но transport-слой пока не подключён).

## Быстрый старт
```bash
cd inn_checker_bot
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env
```

Заполните `.env`, затем запустите:
```bash
python bot.py
```

## Переменные окружения
Обязательные:
- `TELEGRAM_BOT_TOKEN` — токен Telegram-бота.
- `DADATA_API_TOKEN` — API-токен DaData.

Опциональные:
- `DADATA_API_SECRET` — секрет DaData (если используется).
- `OPENAI_API_KEY` — ключ OpenAI (зарезервирован, в runtime не используется).
- `OPENAI_MODEL` — модель OpenAI (по умолчанию `gpt-4.1-mini`).
- `REQUEST_TIMEOUT_SECONDS` — таймаут HTTP-запросов, по умолчанию `10`.

## Основные возможности
- Нормализация ИНН (оставляются только цифры).
- Проверка ИНН на корректность контрольной суммы для 10- и 12-значных ИНН.
- Поиск по ИНН в DaData.
- Короткий и безопасный пользовательский ответ при ошибках запроса.
