# Skill: normalize

## Назначение
Нормализовать обозначения и поля для последующей валидации.

## Правила
1. Не менять регистр.
2. Выделять префикс/номер/суффикс только из подтверждённых частей.
3. Неподтверждённые части оставлять пустыми.
4. Числовые поля (`d`, `D`, `B`, `M`) приводить к числу с `coerce`.
5. Интерфейс распределять только в `ГОСТ` или `ISO`.

## Выход
Нормализованные записи с единым набором полей для GOST/ISO.
