# Skill: validate

## Назначение
Подтвердить аналоги и размеры по источникам истины.

## Шаги
1. Сопоставить пары через `REF_GOST_ISO_размеры`.
2. Проверить совпадение геометрии и исполнения по `MASTER_PRODUCTS`/брендовым данным.
3. При подтверждении заполнить поля аналога.
4. При отсутствии подтверждения записать `NO DIRECT EQUIV`.
5. Удалить дубли по ключу: `Бренд + префикс + номер + суффикс + Аналог`.

## Выход
Провалидированные строки для листов GOST и ISO.
