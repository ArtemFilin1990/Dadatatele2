# Skill: report

## Назначение
Проверить качество выгрузки и сформировать итоговый отчёт.

## Чек-лист
1. Нет пустого `номер` в GOST/ISO.
2. Размеры числовые.
3. `NO DIRECT EQUIV` только для неподтверждённых прямых аналогов.
4. Нет дублей.
5. Листы и порядок колонок соответствуют спецификации.

## Метрики отчёта
- строк в GOST
- строк в ISO
- кодов в PREFIX_DICT
- кодов в SUFFIX_DICT
- строк с `NO DIRECT EQUIV`
- основные конфликты данных
